# Datasets Documentation

This document provides information about the datasets used in the Multiple Disease Prediction System. The dataset is divided into two main directories: raw and processed data.

## Raw Data
The `raw_data` directory contains the original, unprocessed datasets. This data is collected from various sources and is used for initial analysis and pre-processing.

- **Directory Path:** `data/raw_data/`
- **Contents:** 
  - Original dataset files in various formats (CSV, JSON, etc.)
  
## Processed Data
The `processed_data` directory contains datasets that have undergone cleaning, transformation, and feature engineering. This data is used for training and testing machine learning models.

- **Directory Path:** `data/processed_data/`
- **Contents:** 
  - Cleaned and transformed dataset files (CSV)
  - Data description and metadata files (if available)  

## Notes
- Ensure that sensitive information is anonymized in the datasets.
- Regular updates should be made to reflect new data collection and processing techniques when applicable.
